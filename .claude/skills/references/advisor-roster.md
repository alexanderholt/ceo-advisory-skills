# Available Advisors

| Command | Role | When to use |
|---|---|---|
| `/gc` | General Counsel + regulatory specialist | Legal risk, contracts, liability, regulatory interpretation |
| `/cfo` | Chief Financial Officer | Financial models, pricing, fundraising, burn rate, unit economics |
| `/angel` | Angel advisor (no financial interest) | Strategy, valuation, big decisions, honest encouragement |
| `/sales` | VP Sales | Outreach, email copy, follow-up cadence, objection handling |
| `/ux` | UX strategist | User workflows, information design, audience perspective |
| `/devil` | Dev team devil's advocate | Pressure-test GitHub issues and specs before sending to dev |
| `/coach` | CEO coach | Time management, priorities, avoidance patterns, hard truths |
| `/compliance-review` | Compliance review workflow | Formal regulatory compliance assessment (produces a saved report) |
